what you get
everything you need,
nothing you don't

flock does a few things and does them really well. no bloat, no settings rabbit holes, no "coming soon" features.

🪟

panes that just work

they tile themselves. 1 pane fills the screen, 4 make a grid. split, maximize, navigate. it figures it out.

🤖

agent mode

throw tasks at a kanban board. watch agents pick them up and work through them in real time, with a live action timeline.

⌨️

command palette

⌘K opens everything. new panes, themes, layouts, broadcast mode. if you've used raycast you know the vibe.

📡

broadcast mode

type once, every pane hears it. useful when you want all your claudes to know something at the same time.

🔍

find everywhere

search inside one pane or across all of them. ⌘F for one, ⌘⇧F for all. simple.

💾

session restore

close the app, open it later, everything's still there. your layout, your panes, right where you left them.

🎨

themes

because life's too short for one color scheme. switch from the command palette whenever the mood strikes.

🔔

notifications

flock taps you on the shoulder when an agent finishes something. go get coffee, it'll let you know.

under the hood
native swift.
no electron. ever.

each pane is a real terminal powered by SwiftTerm. agent mode talks to claude code over stream-json and parses every event in real time. it's fast because it's not pretending to be a website.

Swift 5.9 AppKit SwiftTerm macOS 13+ stream-json zero web views
